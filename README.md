🏅 Athlete Performance Analytics Dashboard

An interactive Power BI dashboard designed to analyze Olympic athlete performance across countries, sports, and individual athletes.

📌 Project Overview

This dashboard provides insights into Olympic performance using historical athlete and medal datasets. Users can explore trends at three levels:

Global Overview
Country-Level Analysis
Individual Athlete Profiles

The project demonstrates data modeling, DAX calculations, interactive visualizations, and dashboard storytelling in Power BI.


🚀 Features

Global Overview Dashboard:
-Total Athletes Analysis
-Total Medal Count
-Medal Win Percentage
-Country Participation Analysis
-Top Performing Nations
-Gender Distribution
-Geographic Medal Visualization

Country Performance Dashboard:
-Country-wise Medal Distribution
-Medal Trends Over Time
-Sport-wise Medal Analysis
-Gender-Based Medal Comparison
-Athlete Drill-Down Table

Athlete Profile Dashboard:
-Individual Athlete Statistics
-Gold, Silver, and Bronze Medal Summary
-Medal Timeline
-Athlete Demographics
-Performance Summary
